Bobbin Tension Test & Adjustment Summary

1. Sag Test Purpose

  • Verify proper bobbin tension.
  • Standard: Bobbin should slightly drop when the line is gently pulled (free fall = too loose).
  • Timing: Perform before installing tail fiber.

2. Adjustment Method

  • Tool: Use a precision flat-head screwdriver.

  • Direction:

    • Clockwise â†’ Increase tension
    • Counterclockwise â†’ Decrease tension

  • Key Notes:

    • Adjustment screw is highly sensitive â€“ make minute adjustments.
    • Immediately perform a “sag test” after each adjustment.
    • Correct State:
      • Bobbin hangs without natural sag.
      • Drops moderately when gently pulled.
